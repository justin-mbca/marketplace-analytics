# Maine Health Insurance Marketplace Analytics

## Overview
This repository contains data analysis and visualization assets for Maine's CoverME.gov marketplace. It is designed as a portfolio project demonstrating end-to-end analytics: data generation, validation, transformation, and Tableau dashboarding.

## Key Analyses
1. **Enrollment Trends**: Monthly enrollment by county and demographic
2. **Disparity Analysis**: Equity metrics across population groups
3. **Consumer Experience**: Application journey and support metrics
4. **Regulatory Reporting**: Compliance tracking framework

## Setup & Usage
1. Generate sample data: `python data/synthetic_generation.py`
2. Run the ETL pipeline: `python scripts/etl_pipeline.py`
3. Open [notebooks/analysis.ipynb](notebooks/analysis.ipynb) for detailed analysis

## Methodology
- Disparity Index calculated per CMS health equity guidelines (simplified for demo)
- Data validated using automated quality checks
- Visualizations follow accessibility best practices
